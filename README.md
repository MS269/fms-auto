# fms-auto

기부물품관리시스템 접수등록 자동화

---

## 사용법:

1.  excel/list.xlsx 에 리스트 작성

    > EX)
    > |shop|date|amount|cost|
    > |:--:|:--:|:----:|:--:|
    > |a1|0101|10|10000|
    > |b10|0111|100|100000|
    > |c1|1101|100|100000|
    > 유의: 날짜 꼭 4자리수로!

2.  main.py 실행

    > 유의: 실행중 화면 밖으로 마우스 옮기면 중단

3.  한 개 끝나면 확인하고 Enter 입력

---
